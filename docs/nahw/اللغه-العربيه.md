\    <h1 class="text-3xl font-bold text-slate-800 leading-tight">أقسام الكلمة (النحو)</h1>

  </div>



  <div class="bg-white border border-slate-100 rounded-2xl p-8 shadow-sm space-y-6">

\    

\    <p class="text-lg text-slate-600 leading-relaxed">

\    تبنى اللغة العربية على كلمات ثلاث، ولا يخرج أي كلام نتحدث به عن هذه الأقسام الثلاثة وهي: <span class="font-bold text-indigo-600">الاسم، والفعل، والحرف</span>.

\    </p>



\    <div class="bg-amber-50/60 border-r-4 border-amber-500 rounded-l-xl p-4 my-4">

\    <div class="flex items-center gap-2 mb-1">

\    <span class="text-amber-600 text-lg">💡</span>

\    <h4 class="font-bold text-amber-950 text-base">قاعدة ذهبية</h4>

\    </div>

\    <p class="text-slate-700 text-sm leading-relaxed">

\    معرفة نوع الكلمة هو الخطوة الأولى والأساسية للإعراب الصحيح. إذا أخطأت في تحديد نوع الكلمة، سيكون إعرابك خاطئاً بالضرورة.

\    </p>

\    </div>



\    <div class="grid grid-cols-1 md:grid-cols-3 gap-4 mt-6">

\    

\    <div class="border border-slate-100 bg-slate-50/50 rounded-xl p-4 hover:border-indigo-100 transition-colors">

\    <div class="w-8 h-8 rounded-lg bg-indigo-100 text-indigo-600 flex items-center justify-center font-bold mb-3">١</div>

\    <h3 class="font-bold text-slate-800 text-lg mb-1">الاسم</h3>

\    <p class="text-slate-500 text-sm leading-normal">ما دل على معنى في نفسه ولم يقترن بزمن معين. مثل: <span class="text-slate-800 font-semibold">كتاب، أحمد، شجرة</span>.</p>

\    </div>



\    <div class="border border-slate-100 bg-slate-50/50 rounded-xl p-4 hover:border-teal-100 transition-colors">

\    <div class="w-8 h-8 rounded-lg bg-teal-100 text-teal-600 flex items-center justify-center font-bold mb-3">٢</div>

\    <h3 class="font-bold text-slate-800 text-lg mb-1">الفعل</h3>

\    <p class="text-slate-500 text-sm leading-normal">ما دل على حدث وقع في زمن معين (ماضٍ، مضارع، أمر). مثل: <span class="text-slate-800 font-semibold">قرأ، يكتب، اسمع</span>.</p>

\    </div>



\    <div class="border border-slate-100 bg-slate-50/50 rounded-xl p-4 hover:border-rose-100 transition-colors">

\    <div class="w-8 h-8 rounded-lg bg-rose-100 text-rose-600 flex items-center justify-center font-bold mb-3">٣</div>

\    <h3 class="font-bold text-slate-800 text-lg mb-1">الحرف</h3>

\    <p class="text-slate-500 text-sm leading-normal">لفظ لا يظهر معناه كاملاً إلا إذا وضع مع غيره في جملة. مثل: <span class="text-slate-800 font-semibold">في، من، على</span>.</p>

\    </div>
